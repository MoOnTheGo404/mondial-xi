# Model card — Kickoff Atlas match forecaster

Machine-readable version: `ml/artifacts/model_card.json` (regenerated by
`make evaluate`; includes git commit + exact counts).

## Task
Pre-match probabilities (home/draw/away) and scoreline distributions for
senior men's international football.

## Training data
CC0 international results (see data card). Fit window 1980–2018 (~30k
matches); ratings warm up from 1872. No player, market, weather or ranking
inputs.

## Architecture
- Champion: `HistGradientBoostingClassifier` (depth 4, lr 0.06, 400 iters,
  L2 1.0, min-leaf 60) over 17 chronological features, with per-class
  isotonic calibration fitted on the 2019–2022 validation window.
- Companions in the serving bundle: Dixon–Coles goal model (scorelines,
  scenario tilts), Elo-logistic (baseline display), tuned Elo state.

## Evaluation (chronological; see ml/artifacts/metrics.json)
Untouched test 2023 → cutoff (n=3,689): log loss 0.8701, RPS 0.1687,
Brier 0.5106, accuracy 60.8%, ECE 0.0137. Baselines: frequency 1.0543,
Elo-logistic 0.8669, Dixon–Coles 0.8663 (test log loss). Selection was
pre-registered on validation, where the calibrated GBM was best (0.8555);
DC's marginal test edge is reported honestly.

## Intended use
Editorial forecasting, scenario exploration, tournament simulation,
education. **Not** betting advice; no odds or staking outputs.

## Limitations & failure modes
- Team-level only: cannot react to breaking injury/lineup news by itself
  (scenario layer requires user input and is labeled as such).
- Draw class is hardest; per-class reliability shown on /performance.
- Teams with <30 matches carry rating-prior uncertainty (grade C/D + warning).
- Distribution shift (format changes, new teams) degrades gracefully via Elo
  but is not explicitly modeled.

## Trust & artifacts
